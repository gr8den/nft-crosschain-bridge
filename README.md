# Centralized Bridge for swapping NFTs between blockchains

## Developing

* `yarn`
* configure `.env` based on `.env.example`
* `yarn hardhat node` # private network
* `yarn build` # build contracts

## Testing

* `yarn test`

## Task usage examples

* local: `manual-test-local.sh`
* matic & bsc: `manual-test.sh`
